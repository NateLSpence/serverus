Prerequisites

Set up WordPress and log in

You'll need a host, a domain name, and an updated version of WordPress installed in order to use Serverus. Refer to the resources here- LINK for instructions on this.

If you're not signed in already, head to yourwebsite.com/wp-admin to log in as an administrator.

Please note! As administrator, you'll see the WordPress Admin Toolbar at the top of the webpage- other users won't see this. You'll also be able to see IP addresses and extra buttons on posts for moderation. Blocked/muted users and posts will be visible to you, but not regular uesrs. Otherwise, what you see is what your users will see for the most part.

Setup

Front Page Setup

Like most forums, you can have your front page display news. This will display the latest topics from a single forum (for example, "Server News"), newest first, without any of the posts attached to them.

You'll need to put in a shortcode to enable the Serverus front page if you want it. The simplest form of it looks like this: [srv_frontpage forum_id=#]

To set which forum to display posts from, change the "forum_id=#" to the id number of your preferred forum. You'll find the ID...

There are some additional settings to customize the way the front page works. The following are optional: posts_per_page=5 char_limit=0 show_stickies=false Configuring all these settings in the shortcode would look like this: [srv_frontpage forum_id=12 posts_per_page=7 char_limit=450 show_stickies=true]

Forums Setup

Create a blank page called "forums". Serverus will take care of the rest here. What's important is that you have a page with the slug "forums", so after you create this page, you can edit its title to whatever you want and it won't break anything.

Under the Forums menu, you'll have to create the forums you want, along with categories to put them under. All forums require a category, even if everything is under one category.

Widgets

Login/Registration Widget

Drag the (Serverus) Login Widget to the Login Location. Now you and your users can log in.

Recent Topics Widget

This widget displays a list of recent posts made by your users.

Other Widgets

The (bbPress) Forums Statistics widget and (bbPress) Forums List widgets might be useful as well. Text widgets work for any static information you want visible (Server IP, links, etc). For any third-party widgets, paste the code into a Text widget.

Users

Enable Registration

To open your forums to the world, you'll have to enable registration in order for users to make their own account. Make sure "Anyone can register" is checked under Settings. Alternatively, you can opt to add users manually via the Users menu.

Modify Users

There are two places you can edit a user's information- the Users menu from the WordPress menus, or by clicking on the user's name in the forums and viewing their profile. These are basically the same, but as an administrator you might find them both useful. In the WordPress menus, you can see all of the users in a list, instead of digging through posts to find a specific person. Making quick tweaks might be easier from the forum profile view.

Moderation

If you've installed the bbPress Moderation plugin, this is a pretty simple process. Individual posts have buttons for moderation, and you can set a user to "Blocked" as their forum role to ban them. Setting a user to "Moderator" gives them the ability to edit other users/posts and see a minimal version of the WordPress administration menus.

Customizing

Time to customize! Add a header image and text, a background image, and choose a color scheme that suits your fancy.

Serverus is simple, but you can dig into the color scheme if you want to. The best method is to choose a color scheme that is closest to what you need already, and then tweak it from there. If you're itching to change something, but it doesn't have an option, you can always override the default CSS .
