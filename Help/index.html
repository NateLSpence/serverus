Setting up Serverus

Prerequisites

Set up WordPress and log in

You'll need a host and domain name with an updated version of WordPress installed in order to use Serverus.

Your web host might have an option to install WordPress automatically, or you may have to download and install it yourself.

Once you have WordPress up and running, log in.

As administrator, you'll see the WordPress Admin Toolbar at the top of the webpage- other users won't see this. You'll also be able to see IP addresses and extra buttons on posts for moderation. Blocked/muted users and posts will be visible to you, but not regular users. Otherwise, what you see is what your users will see for the most part.

Installation

Theme Installation

Go to the Admin page and navigate to Appearance->Themes. Click "Add New" at the top of the page.

Click "Upload Theme" at the top of the page.

Click "Choose File" and navigate to the serverus-theme.zip file. Upload it, and click "Install Now".

Now activate the theme.

Plugins

Required and Recommended Plugins

Upon activating Serverus, you will see a prompt asking you to install some plugins. You can just select them all and tell them to install, or you can pick and choose. If you've dismissed the prompt, you can still view these plugins under "Appearance->Install Plugins".

bbPress is the forum software. It's required for Serverus to run.

bbPress Moderation makes it easier to prevent spam by making a user's first post require approval by a moderator.

bbPress Signature adds the ability for users to put a signature in their posts.

Easy Bootstrap Shortcodes gives you more options for pretty formatting when making additional pages.

Author Avatars List lets you display a widget showing all of your forum's users. Common use would be to set the Avatar size to 32 or 16, and show the following roles: Keymaster, Moderator, and Participant.

bbPress Admin Bar Addition gives Admins and Moderators a handy menu on the top Admin bar to better manage the forums.


Feel free to browse around for more plugins! If there's something you want to do with your site or forum, there's usually a plugin that makes it easy.

Setup

Front Page Setup

Like most forums, you can have your front page display news. This will display the latest topics from a single forum (for example, "Server News"), newest first, without any of the replies attached to them.

Serverus uses a shortcode to determine which forum to get news from, and to set some options:

[srv_frontpage forum_id=# posts_per_page=5 char_limit=0 show_stickies=false]

Firstly, forum_id=# is the ID of the forum to display. Browse to a forum to find its ID (only Admins and Mods see them). Use a number here; no quotes or symbols.

Then, posts_per_page=5 decides how many posts to display on the front page before you have to click "Next".

The option char_limit=0 is the maximum length, in characters, to display for each post. 0 means there is no limit.

Lastly, show_stickies=false decides whether or not to display "sticky" threads in your news page. Use true or false here.


An example of a customized shortcode might look like this:

[srv_frontpage forum_id=15 posts_per_page=20 char_limit=450 show_stickies=false]

In this example, our front page will display posts from the forum with the ID of 15. It will display a maximum of 20 posts, but only the first 450 characters from each. Sticky threads won't be displayed.

Forums Setup

Under the Forums menu, create the forums you want, along with categories to put them under. All forums require a category, even if everything is under one category.

Menus

This is no different from your typical WordPress menus. If your pages aren't showing up in your Nav bar and you're not sure how to add them, read this guide. To change the colors of the Nav bar, check out the "Customizing" section below.

Users

Enable Registration

To open your forums to the world, you'll have to enable registration in order for users to make their own account. Make sure "Anyone can register" is checked under Settings. Alternatively, you can opt to add users manually via the Users menu.

Modify Users

There are two places you can edit a user's information- the Users menu from the WordPress menus, or by clicking on the user's name in the forums and viewing their profile. These are basically the same, but as an administrator you might find them both useful. In the WordPress menus, you can see all of the users in a list, instead of digging through posts to find a specific person. Making quick tweaks might be easier from the forum profile view.

Moderation

If you've installed the bbPress Moderation plugin, this is a pretty simple process. Individual posts have buttons for moderation, and you can set a user to "Blocked" as their forum role to ban them. Setting a user to "Moderator" gives them the ability to edit other users/posts and see a minimal version of the WordPress administration menus.

Customizing

The "Customize" screen

Time to customize! Add a header image and text, a background image, and choose a color scheme that suits your fancy.

Serverus is simple, but you can dig into the color scheme if you want to. The best method is to choose a color scheme that is closest to what you need already, and then tweak it from there. If you're itching to change something, but it doesn't have an option, you can always override the default CSS.

Single pages

If you want to use some of the slick Bootstrap elements, make sure you've got the Easy Bootstrap Shortcodes plugin installed.

Disable the sidebar, nav bar, or foreground

If you want something like a landing page, you may want to make a page that doesn't have the sidebar, the top navigation bar, or the foreground. To do this, create or edit the page in question, and find the "Template" dropdown in the "Page Attributes" box on the right. Choose the template you want and save the page.

Troubleshooting

I can't get to any of my pages.

If you're getting an error along the lines of "404 page not found" then you probably need to enable Pretty Permalinks and mod_rewrite. If you're having trouble with this, contact your web host's support.

I just see a white screen!

This is what WordPress does when it breaks. What is the very last thing you did before you saw the white screen? If you just enabled the theme, or one of the plugins, your best bet is to follow WordPress's help guide. After that, double-check that you've updated WordPress, bbPress, and Serverus itself to their latest versions. Double-check the Prerequisites section. Lastly, remember that bbPress must be activated for Serverus to work properly. Disabling it while Serverus is activated might cause issues.
